A generic class for handling MapReduce resonses. Not required to use but is helpful if returning more than one value. MapReduce returns a collection of objects that are fieldSelectionExpando - value pairs. The value can be a single value, or more likely a document response. If you are getting back a single int you could declare MapReduceResponseGeneric<int%gt;, if its a more complex type it could be MapReduceResponseGeneric<myclass>
